# **Thought Process**

We can solve this problem in many ways. One approach could be creating a dictionary of each character with their incremented values and then retrieving these values.

Another approach can be converting each character in ASCII and incrementing the value by **1**.

Here, we will convert each letter in a string to ASCII and then increase it by **1** and again convert the ASCII to the character.

To solve this problem, we can create two separate functions, one to encrypt the character and one to determine if the character is okay to encrypt or not.

Steps to work in **encrypt function**,

1. If the passed character is alphabet `'z'`, assign `'a'` to the `secret` variable.
2. If the passed character is alphabet `'Z'`, assign `'A'` to the `secret` variable.
3. Else, convert the character to ASCII value using the inbuilt `ord()` function.
4. Then, add **1** to the ASCII value.
5. Again convert the ASCII value to character using the inbuilt `chr()` function.
6. Then, assign the converted character to the `secret` variable.
7. Return the secret character variable.

Similarly, we have another **main runner function**. Here're the steps we need to follow,

1. Initialize the empty string to store the secret string.
2. Loop through each character in the given string.
3. Check if the character is an alphabet.
4. If the character is an alphabet, call the encrypt function by passing the current character as a parameter and append the return value to the secret string.
5. If the character is not an alphabet, append the character as it is to the secret string.
6. Return the secret string.

```python
def encrypt(character):
    if character == 'z':
        secret = 'a'
        
    elif character == 'Z':
        secret = 'A'
    else:
        secret = chr(ord(character)+1)
    return secret
def main_runner(string):
    result = ''
    for char in string:
        if char.isalpha():
            result += encrypt(char)
        else:
            result += char
    return result
encrypted = main_runner('i am lazy')
print(encrypted)
encrypted = main_runner('I AM LAZY')
print(encrypted)
```

# 11- Bulls and Cow Problem

This problem is based on the popular game **Bulls and Cows**.

Suppose you and your friend are playing the game. In this game, you need to write down a secret number and make your friend guess the number.

After guessing, you'll give him some hints. You will tell your friend,

- The total number of digits in the guessed number whose positions in that number match the position of the same digits in the secret number. This total is known as the count of **bulls**.
- The total number of digits in the guessed number whose positions in that number don't match the position of the same digits in the secret number. This total is known as the count of **cows**.

Let's look at this with an example. Suppose the secret number is **1807**, and your friend's guess is **7810**.

In your friend's guess, only the position of the digit **8** matches the position of the **8** in the secret number. On the other hand, the position of the other three digits don't match at all.

Hence, this is what you can tell about your friend's guess:

- **bulls = 1** (correct digit positions)
- **cows = 3** (incorrect digit positions)

Now, your friend needs to rearrange the digits that are **cows**, to make those digits **bull**.

So, write a Python function that takes a secret number and your friend's guess as parameters. This function should return the hint for your friend's guess.

The hint format should be `xAyB` where `x` is the number of **bulls** and `y` is the number of **cows**.

**For example**

```
secret = 1807
guess = 7810
```

A partition of a number `n` is an unordered sum of one or more numbers which totals n. For example, the partitions of the number **4** are:


Since partitions are unordered, the sums `1 + 1 + 2 = 1 + 2 + 1 = 2 + 1 + 1 = 4` are considered the same partition.

- Define a function named `calculate_partitions()` with a single argument `n`.
- Inside the function, calculate the number of partitions of the natural number `n`.

```python
def calculate_partitions(n):
    # Create a table to store the number of partitions for each number from 0 to n
    partitions = [0] * (n + 1)

    # There's one partition for 0 (the empty set)
    partitions[0] = 1

    # Calculate the number of partitions for each number from 1 to n
    for i in range(1, n + 1):
        for j in range(i, n + 1):
            partitions[j] += partitions[j - i]

    # The last element of the partitions list will contain the result
    return partitions[n]
```
